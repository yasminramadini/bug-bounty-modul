OS Command Injection

A. Simple command injection

  1. Misalnya sebuah web memiliki fitur check stok produk yang mengirim parameter ID ke backend
  2. Cobalah masukkan perintah berikut:
    1|whoami
    Jika vulner, maka akan menghasilkan nama user saat ini

B. Blind OS Command Injection dengan time delay

  1. Misalnya suatu web menyediakan submit feedback.
  2. Pada kolom email misalnya, kita bisa masukkan command:
    email=x||ping+-c+10+127.0.0.1||
    Jika vulner, maka akan delay 10 detik
